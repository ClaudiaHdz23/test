Instrucciones:

  1. Crear un servicio que se encargue de hacer peticiones GET
  2. Crear componente (customSelect) al que se le pasará información
  3. Hacer petición GET dentro del servicio creado para obtener archivo json: ./assets/window.json
  4. Dentro del window.json obtenido a través del servicio, filtrar elemento con "type": "select" del arreglo "controls" y usar su propiedad "style" como estilo de un elemento html select dentro del componente creado (customSelect)
  5. Hacer petición GET dentro del servicio creado para obtener archivo json: ./assets/formats.json
  6. Llenar select de customSelect con arreglo de formatos, mostrando sólo aquellas cuyo atributo "type" sea igual a 1
  7. Crear un formulario dentro del componente "customSelect" (de preferencia usar reactive forms) que se llene con los campos del objeto seleccionado [id, format, formatDisplay, type]
  8. Permitir modificar solamente campo "formatDisplay" (los demás campos deben permanecer inhabilitados)
  9. Crear botón que permanezca inhabilitado hasta que se modifique el valor de "formatDisplay" y que capture los valores de la forma una vez sea habilitado
  10. Mostrar valores del formulario
